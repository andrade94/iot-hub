Design Notes
Index: Metrics as translated badges Shows "Temperature", "Humidity" not "temperature", "humidity". Uses the metric_* translations.
Index: Valid ranges inline Compact display of min-max per metric. For models with many metrics shows "4 ranges" instead.
Form: Supported Metrics as checkboxes Pre-defined list of known metrics with translated labels. Plus a "+ Custom..." option for new metrics. Selected metrics determine the Valid Ranges section below.
Form: Dynamic Valid Ranges A row per selected metric with min/max inputs. Rows appear/disappear as metrics are toggled. The metric name + unit shown as label.
Show: Metrics & Ranges table Full table with metric name, key, min, max, unit. The key info for understanding sensor capabilities.
Show: Recipes section All recipes that use this sensor model, with rule count + device count. Click navigates to recipe detail.
