Design Notes
Cards instead of table Sites usually have 1-3 gateways — cards give more visual weight than table rows. Each card shows model, serial, status, device count, type, last seen.
Primary vs Addon distinction Primary gateway handles all devices. Addon is for coverage extension. Shown as badge + description text.
Show page: connected devices Table of all devices using this gateway — name, model, zone, status, last reading. Click navigates to device detail.
Edit capability (NEW) Currently missing — adding update route + dialog. Model, serial, and addon toggle editable.
Delete with device warning Shows device count and warns that devices will lose their gateway connection. Doesn't block — just warns.
ChirpStack ID shown Technical reference for the LoRaWAN server integration. Shown in detail sidebar, not prominent.
