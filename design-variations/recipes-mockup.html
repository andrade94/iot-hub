Design Notes
Index: Device + Override counts Shows how many devices use each recipe and how many site overrides exist. Answers "is this recipe actively used?"
Index: Rule pills Compact severity-colored pills showing metric + condition + threshold. Scannable at a glance.
Show: Devices section NEW — Shows which devices use this recipe, with site name, zone, and online status. Answers "what will be affected if I edit this?"
Show: Overrides with diff view Side-by-side default vs override values. Changed thresholds highlighted in primary color, unchanged show "no change". Clear visual diff.
Show: Delete override NEW — "Remove" button per override. Reverts site to default recipe thresholds.
Show: Quick links sidebar Links to alert rules, sensor model page, and module page. Connects the recipe to its related entities.
