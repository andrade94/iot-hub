Frequently Asked Questions

How does multi-tenancy work?

Manual scoping with org_id and site_id columns. Two middleware enforce isolation: EnsureOrganizationScope binds the current org, and EnsureSiteAccess verifies site-level permissions via the user_sites pivot table.

What authentication does the platform use?

Laravel Fortify for web (session-based) with 2FA support, and Laravel Sanctum for the mobile API (token-based). Spatie Laravel Permission manages the 7 roles and 29 permissions.

How are real-time updates delivered?

Laravel Reverb provides the WebSocket server. Three broadcast events push live data: SensorReadingReceived, AlertTriggered, and NotificationCreated. The frontend consumes these via Laravel Echo.

What database does it use?

PostgreSQL 16 with TimescaleDB extension for time-series data. Raw sensor readings are compressed after 7 days (90%+ reduction). Continuous aggregates pre-compute 15m, 1h, and 1d rollups. Redis caches latest readings for sub-millisecond dashboard loads.

Is there a mobile app?

Yes. The Expo SDK 54 (React Native) companion app lives in a separate iot-expo repository. It connects via Sanctum API tokens with 26 endpoints. Push notifications are sent via Expo Push API.
